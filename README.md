# Mowla Web - Modern Yazılım Şirketi & Portföy Sitesi

Modern, profesyonel ve güven veren bir yazılım şirketi ve kişisel portföy web sitesi.

## Teknolojiler

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **next-intl** - Internationalization (Türkçe/İngilizce)

## Özellikler

- ✅ Responsive tasarım (mobil, tablet, desktop)
- ✅ SEO uyumlu
- ✅ Çok dilli destek (TR/EN)
- ✅ Modern animasyonlar
- ✅ Koyu tema
- ✅ Minimal ve profesyonel tasarım

## Kurulum

```bash
npm install
npm run dev
```

Site [http://localhost:3000](http://localhost:3000) adresinde çalışacaktır.

## Sayfalar

- **Ana Sayfa** - Hero section ve genel bakış
- **Hizmetler** - Web geliştirme, mobil uygulama, bakım, dijital reklam
- **Hakkımda** - Kişisel hikaye ve journey
- **İletişim** - İletişim formu ve sosyal medya linkleri

## Marka Kimliği

- **Ana Renk**: Mavi tonları (Primary)
- **Vurgu Rengi**: Mor tonları (Accent)
- **Font**: Inter (sans-serif), Poppins (display)
